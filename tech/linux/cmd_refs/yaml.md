# yaml

###### Scalar block notation
- `|` literal style, line breaks preserved
- `>` folded style, line break is folded to space unless it ends in an empty or a more indented-line

###### Block chomping indicator
- `+` Keep, final line break and trailing empty lines are kept
- `-` Strip, final line break and trailing empty lines excluded
- `<none>` Clip, final line break is preserved but trailing empty lines excluded
