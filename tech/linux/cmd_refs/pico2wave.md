# pico2wave

## Use pico2wave to convert text to speech, output to a Wav file and play it
```bash
pico2wave -l=en-GB -w=out.wav "The quick brown fox jumped over the lazy dog." && aplay out.wav
```

### Options for languages

* `en-US`   English
* `en-GB`   Great Britain
* `es-ES`   Spanish
* `fr-FR`   French
* `it-IT`   Italian
